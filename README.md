# page-aiyouxi-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常作为静态资源使用，不依赖于特定域名或后端服务。仓库内容以页面文件为主，适合用于演示、学习或作为其他项目的资源引用。

## 目录说明

```
.
├── pages/          # 存放各 HTML 页面文件
├── assets/         # 公共资源（如样式、脚本、图片等）
└── README.md       # 本文件
```

- `pages/`：每个子目录或单独文件对应一个独立页面，命名清晰，便于查找。
- `assets/`：存放页面间共享的静态资源，减少重复引用。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可直接在浏览器中打开运行。
- 页面内可能引用本仓库 `assets/` 目录下的资源，请确保相对路径正确。
- 归档页面不包含任何外部依赖的跟踪代码、广告或营销内容。
- 页面内容保持简洁，聚焦于功能或展示目的。

## 维护说明

- 本仓库主要供个人或协作归档使用，不提供实时更新保证。
- 如发现问题或需要改进，欢迎提交 Issue 或 Pull Request。
- 添加新页面时，请放入 `pages/` 目录，并保持命名规范。
- 请勿在页面中嵌入任何指向具体网站、域名或带有推广性质的链接。

## 使用方式

1. 克隆或下载本仓库。
2. 直接打开 `pages/` 下任意 HTML 文件即可预览。
3. 也可将整个仓库部署到任意静态服务器上使用。
